# Guess The Number

A simple JavaScript project which lets the user to guess a randomly generated number in 10 attempts.

## Technologies Used

- HTML
- CSS
- Javascript

## What I learned

- DOM manipulation using JavaScript.
- Structuring logic for appliaction by breaking it into smaller functions.
- Managing state of the game using variables and conditions.
- Updating UI based on user interactions.

## Note

The project was built while following a JavaScript tutorial, and I coded along with the instructor as a part of the learning process. AI assistance was used for the generation of CSS whereas the HTML and JavaScript was written by me during the tutorial. The project represents my understanding and implementation of the concepts covered.
